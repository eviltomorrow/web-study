# 网页的构造块

汇总

- 一个网页主要由三种成分构成：文本内容、对其他文件的引用和标记。
- HTML 标记由元素、属性和值构成。
- 通常全部使用小写字母编写 HTML（DOCTYP 是一个例外），用引号包住属性值。
- 为文件和文件夹命名时全部采用小写字母，词与词之间用短横线分割，而不要用空格或者下划线。
